# Ticket Booking System

A C++ console application for booking tickets for buses, trains, and flights.

## Features

- Search and book tickets by departure/destination cities
- Support for buses, trains, and flights with real-time availability and pricing
- Cancel tickets
- Check vehicle status and list all available options
- View reserved tickets
- Data persistence using a text file
- Object-oriented design with inheritance and polymorphism

## Sample Routes

- Buses: Delhi to Goa (₹1500), Mumbai to Goa (₹1200), Delhi to Mumbai (₹800)
- Trains: Delhi to Goa (₹2500), Mumbai to Goa (₹2200), Delhi to Chennai (₹3000)
- Flights: Delhi to Goa (₹8000), Mumbai to Goa (₹7500), Delhi to Bangalore (₹6000)

## How to Compile and Run

1. Ensure you have a C++ compiler installed (e.g., g++ from MinGW or Visual Studio).
2. Compile the code: `g++ main.cpp -o booking_system.exe`
3. Run the executable: `./booking_system.exe` (or `booking_system.exe` on Windows)

## Usage

- Choose option 1 to search for tickets: Enter from/to cities and type, view options with prices, then book.
- Other options for cancellation, status check, etc.

## Technologies Used

- C++ STL (vectors, strings, fstream)
- OOP principles (classes, inheritance)
- File I/O for data persistence
